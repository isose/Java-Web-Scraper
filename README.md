# Java-Web-Scraper
Web scraper build with Java. It's purpose is to to go on Reddit and scrape games that are on sale and are also free!

## How does it work?
We are using Java to create the models and using the Jsoup framework to scrape data. Java FX is also used to create a nice
appealing UI.
